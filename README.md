This project is a simple and functional e-commerce store web application where users can browse products,
add items to their shopping cart, and view the total cost of selected items.
It provides a smooth user experience for basic online shopping needs.

🔍 Features
Product Listing: Users can view a selection of items available for purchase.

Add to Cart: Each product includes an "Add to Cart" button. Users can add one or more products to their cart.

Multiple Selections: Users can add multiple different products or multiple quantities of the same product to the cart.

Shopping Cart: The cart displays all selected items along with their individual prices.

Total Price Calculation: The total price of all items in the cart is automatically calculated and displayed.

Interactive UI: The interface updates dynamically as users interact with the store.
